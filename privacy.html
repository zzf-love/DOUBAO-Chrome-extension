隐私政策

最后更新：2025-05-21

AI Screenshot Explainer（以下简称"本扩展"）由个人开发者发布。 本隐私政策说明本扩展如何处理您的数据。

收集的数据

  • 截图内容：您主动框选的网页区域截图（PNG 图像），仅在您触发分析时产生。
  • API Key：您填写的火山方舟 API Key，仅存储在本地 chrome.storage.local，不离开您的设备，除非随请求发送至您配置的 API 服务端点。
  • 设置项：模型名称、Endpoint URL、默认 Prompt，同样仅存储在本地。

数据使用方式

  • 截图和 Prompt 会发送至您在设置中配置的 API 服务（默认为火山引擎方舟平台）用于内容解析。
  • 本扩展不向任何服务器收集、上传或共享除此之外的任何数据。
  • 本扩展不追踪您的浏览行为，不使用 Cookie，不进行任何形式的数据分析或广告定向。

第三方服务

截图数据发送至火山引擎方舟平台（volcengine.com）。 您可在扩展设置中更改 API 端点以使用其他兼容服务。 使用第三方服务时，其隐私政策同样适用，请自行查阅。

数据存储与安全

所有设置（包括 API Key）均存储在 Chrome 的本地存储中，不经过任何云同步或第三方服务器。 API Key 以明文存于 chrome.storage.local，受 Chrome 扩展沙箱保护， 其他网页或扩展无法访问。请妥善保管您的设备访问权限。

权限说明

  • activeTab / tabs：获取当前标签页的截图，仅在您主动触发时执行。
  • scripting：在网页上注入框选覆盖层和结果面板的脚本。
  • storage：在本地存储您的设置。
  • contextMenus：在右键菜单中提供快捷入口。

儿童隐私

本扩展不面向 13 岁以下儿童，也不有意收集儿童数据。

变更通知

隐私政策如有重大变更，将在扩展更新日志和本页面中说明。
