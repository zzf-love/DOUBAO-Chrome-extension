# 小红书发布素材

3 个不同角度的文案，按你想塑造的"人设"挑一个用。也可以混着用——发 2-3 篇，每篇一个角度。

---

## 📝 模板 1：痛点共鸣式（推荐 ⭐，转化率最高）

**适合人设**：刷信息流、看英文文章、看图表的"信息工作者"

### 标题（≤ 20 字）

```
看到外文截图都跑去问 ChatGPT？教你 0.5 秒搞定
```

或：

```
我把豆包做成了浏览器框选截图工具，太爽了
```

### 正文

```
真实场景👇

刷推 / 看 Dribbble / 翻外文文档时，遇到一段不认识的英文 / 日文 / 代码，
你是不是会：

❌ 复制到翻译软件粘贴
❌ 截图保存→打开 ChatGPT→上传图片→等转圈
❌ 干脆跳过假装看见过

我嫌烦，自己写了个 Chrome 插件 👉

按 Alt + S 拖拽框选任意区域
0.5 秒后，右上角浮动面板出现 AI 解读
继续滚网页，结果留在面板里慢慢看

✨ 关键特性
- 框选触发，不打断浏览节奏
- 多次框选会累积成卡片，方便对比
- API Key 仅存本地，不经过我服务器
- 接的是火山方舟（豆包），免费额度个人完全够用

🎁 装法
评论区扣"安装"我私你下载链接和教程
也可以直接 GitHub 搜：DOUBAO-Chrome-extension

#chrome插件 #效率工具 #AI工具 #豆包 #程序员 #生产力 #软件推荐
```

## 📝 模板 2：技术 / 独立开发者人设

**适合人设**：想塑造"技术大佬""独立开发者"形象，吸引同好

### 标题

```
独立开发｜花一个周末做了个 Chrome 插件
```

或：

```
我做了个 Chrome 插件，把豆包多模态接成了"网页截图问答"
```

### 正文

```
背景
看 Dribbble、Hacker News 时经常遇到不认识的英文，
切到 ChatGPT 上传图片来回好几步，烦。

想法
能不能像系统截图一样，按个快捷键框一下就直接出答案？

实现
🛠 Chrome MV3 + chrome.scripting.executeScript（按需注入，不要 <all_urls>）
🛠 chrome.tabs.captureVisibleTab + canvas 裁剪
🛠 接火山方舟 Doubao Seed 2.0 Responses API
🛠 浮动面板用 vanilla JS 手撸，无依赖

设计取舍
✔️ activeTab 权限而不是 <all_urls>，更符合 CWS 推荐
✔️ API Key 仅本地存储，不收集任何用户数据
✔️ 默认接豆包，但 Endpoint 可改，理论兼容任何 OpenAI Responses 风格视觉模型

成果
v1.2.0，已提交 Chrome 应用商店审核中
源码 + 安装包：GitHub 搜 DOUBAO-Chrome-extension

下一步想做
- 划词翻译模式
- 自定义快捷 Prompt 模板
- 支持多模型并排对比

欢迎 PR / Issue / Star ⭐

#chrome扩展 #独立开发 #AI #豆包 #火山方舟 #编程 #开源 #manifestv3
```

## 📝 模板 3：教程 / 实测式

**适合人设**：测评博主、效率工具分享号

### 标题

```
实测 | Chrome 上有个免费插件，比 ChatGPT 截图问答快 10 倍
```

或：

```
保姆级教程｜浏览器装个插件，看英文截图直接出中文解读
```

### 正文

```
📌 这个插件能干嘛
在任何网页按 Alt+S 拖拽框选→AI 几秒内出解读
适合看：英文/日文截图、错误信息、图表、代码、术语

📌 用什么模型
火山方舟（豆包）多模态，新用户有免费 token 额度
也可以改 Endpoint 接其他兼容 OpenAI Responses 的视觉模型

📌 安装步骤（无需 VPN）
1️⃣ 下载 zip（评论区扣"插件"我发你网盘链接）
2️⃣ 解压到一个固定文件夹
3️⃣ 浏览器输 chrome://extensions/，打开右上角开发者模式
4️⃣ 点"加载已解压的扩展程序"选刚才那个文件夹
5️⃣ 火山方舟官网申请 API Key（ark- 开头）
6️⃣ 点插件图标填 Key 保存
搞定！

📌 用法
按 Alt+S → 拖拽框选 → 右上角面板出结果
右键或 Esc 取消框选
同一页可以连续框选多次，结果叠卡片

📌 体验对比
ChatGPT：截图→保存→打开→上传→提问→等转圈 ≈ 30s
这个插件：Alt+S→框→松手→等 3s ≈ 5s
做表/读论文/翻外网时差距巨大

📌 隐私
✓ API Key 仅存本地 chrome.storage.local
✓ 截图直接从你的浏览器发到火山方舟，不经过插件作者
✓ 不收集任何浏览数据 / Cookie / 行为日志

#效率工具 #chrome插件 #AI神器 #生产力工具 #豆包 #软件推荐 #ChatGPT替代
```

## 发布时机

- **第一篇预热**：CWS 审核**通过前**发，文末挂"下载链接评论区扣""安装"或私信"（引导互动）
- **第二篇上架**：审核通过后立刻发，标题加"✨已上架 Chrome 应用商店"，正文贴商店链接
- **第三篇复盘**（可选）：上线 1-2 周后发"X 人安装/我学到什么"，做长尾流量

## 防限流小贴士

- 标题不要写"破解""免费""无限"等敏感词
- 正文里 "ChatGPT" 提一次就够了，太多次容易判违规
- 评论区引导话术换花样：扣"安装" / 扣"插件" / 扣"教程" / 私信 1
- 配图加文字水印自己的小红书 ID 防搬运
- 互动率高的评论手动回，前 1 小时数据决定能不能进流量池
